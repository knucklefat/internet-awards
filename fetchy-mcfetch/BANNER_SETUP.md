# Banner Image Setup

## Required Action

Please save the "The Internet Awards" banner image (the one with the orange upvote arrow and pink/orange striped border) to:

```
src/client/public/images/internet-awards-banner.png
```

## Image Details
- The image shows "THE INTERNET AWARDS" in white text
- Features an orange upvote arrow
- Has a pink/orange striped border at the top
- Black background

## Fallback
If the image is not found, the app will automatically fall back to a styled div with gradient background showing the same text.

## After Adding the Image
Run:
```bash
npm run build
npm run deploy
```
